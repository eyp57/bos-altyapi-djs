# Kurulum

### Gereken NPM Modülleri
+ discord.js@14.6.0
+ mongoose@latest
+ fs@latest

### Modül kurulumu

```bash
$ npm install discord.js@14.6.0 mongoose@latest fs@latest
```

### Başlatma
```bash
$ node .
```
veya
```bash
$ node index.js
```
